# MACD Signal Output

Latest signal statistics and dashboard generated by the [MACD Signal Scanner](https://github.com/syinsync/macd-signal-scanner).

## Contents

| Path | Description |
|------|-------------|
| `dashboard.py` | Streamlit dashboard — run with `streamlit run dashboard.py` |
| `results/signal_stats_*.csv` | MACD crossover signal stats — entry/exit dates, return %, hold duration |

## Latest Run

```bash
pip install streamlit pandas numpy
```
